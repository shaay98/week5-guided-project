I have learned a lot about CSS. Using flexbox is my property using CSS so far. I love the way you can move your content around on the page in a very detailed manor that's not to difficult to understand. The layout of the page can result however you expect if you are using the correct tools.

Block elements start in a new line and take up the full width available. They stretch out to the left and right as far as they can. A few examples are H1, P, Table, and Div. Inline elements only take up as much space necessary. A few examples are A, Img, Button, and Span. It is best practice to store inline elements inside of Block elements.
